Punto de Inicio

Réplica del Point of Sale con tablas compartidas

Características Principales

Reutilización de Tablas

No crea nuevas tablas en la base de datos. Extiende los modelos existentes del POS (pos.order, pos.session, pos.config, etc.) agregando campos identificadores.

Interfaz Familiar

La interfaz replica fielmente la del Point of Sale, manteniendo la misma usabilidad y flujo de trabajo que los usuarios ya conocen.

Cómo Funciona

Campos Identificadores:
• x_is_punto_inicio: Marca registros de Punto de Inicio
• x_source_module: Indica el módulo de origen

Marcado Automático:
Todos los registros creados desde Punto de Inicio se marcan automáticamente, sin necesidad de intervención manual.

Filtrado Inteligente:
Las vistas y menús filtran automáticamente para mostrar solo los registros relevantes de cada módulo.

Extensible:
El código está preparado para agregar funcionalidades específicas de Punto de Inicio en el futuro.

Instalación

1. Asegúrate de tener instalado el módulo point_of_sale
2. Instala el módulo Punto de Inicio desde Apps
3. Accede al menú "Punto de Inicio" en la barra superior
4. Crea tu primera configuración de Punto de Inicio
5. ¡Empieza a usarlo como un POS normal!
